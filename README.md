# Handwritten-digit-recognition
This project generates a model that is able to recongnize handwritten digits in a 28x28 format in a gray scale
The data used for training, validation and tests are from MNIST
To use this program please make sure that you have all libraries listed in requierements.txt
To train the model and get some useful graphs about it (confusion matrix, per-class accuracy and training curves), run main.py
