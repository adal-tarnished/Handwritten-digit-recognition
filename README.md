# Handwritten Digit Recognition

An elegant and efficient machine learning model trained on the classic MNIST dataset to recognize handwritten digits (0-9) in a gray-scale 28x28 pixel format.

This project not only trains the model but also generates insightful visualizations (such as confusion matrices, learning curves, and per-class accuracy) to help you evaluate its performance.

---

## Key Features

* **MNIST Standardized:** Designed specifically for 28x28 grayscale images.
* **Comprehensive Evaluation:** Automatically generates:
    * **Training and Validation Curves** (Loss and Accuracy over epochs).
    * **Confusion Matrix** to spot which digits the model confuses most.
    * **Per-Class Accuracy** breakdown.
* **Ready to Run:** Structured for quick setup and execution.
